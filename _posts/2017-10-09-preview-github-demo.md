在项目源代码页面链接前缀那加上

```
http://htmlpreview.github.com/?
```

举个栗子:需要打开的项目页面

```
https://github.com/aisinvon/aisinvon.github.io/blob/master/index.html
```

给加上前缀

```
http://htmlpreview.github.com/?
```

最终链接

```
http://htmlpreview.github.io/?https://github.com/aisinvon/aisinvon.github.io/blob/master/index.html
```
